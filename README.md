PetNation is a web based platform where users can buy or sell their their pets. Users can also find homes for homeless stray animals. The users have to provide basic details about the pet and the pet will be listed on the portal where other users can view the pet's details and contact the pet's current owner.

The application is developed using React Js, Tailwind CSS, Express Js, node Js and MongoDB.
